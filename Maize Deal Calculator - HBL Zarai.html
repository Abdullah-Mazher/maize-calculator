How the True Rate Works

Grain solids never change during drying — only water leaves. As the batch gets lighter, the leftover water becomes a larger share of a smaller total, so dropping moisture by a given number of points removes a slightly larger share of total weight. The true rate deducts that actual weight loss.

True Weight Loss % = 1 − ( (1 − Inbound) ÷ (1 − Standard) )
True Rate = Base × ( 1 − True Weight Loss % − Foreign Matter % )

Reading the result: The premium is money paid to the farmer above the true value of the dry grain we keep. It is not a loss to correct — it is a figure to know, so the premium is a deliberate decision rather than a hidden one.
